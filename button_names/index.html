    What is button alias?¶

    The database of infrared files has a ton of different buttons. Every author of infrared file has a different definition of naming. Thus, the names of buttons are different.

    This list of aliases defines the finite set of button names, which can be used with IRDB.

    Please, use only aliases from this list for every avaliable button

    If you don't see here button or alias you need, contact us via new issue.
